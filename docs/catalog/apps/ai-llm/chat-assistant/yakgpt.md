# YakGPT

> YakGPT is a simple, locally running ChatGPT UI that makes your text generation faster and chatting even more engaging! This hands-free interface provides direct access to OpenAI's GPT models with advanced voice capabilities, including Speech-to-Text via Azure & OpenAI Whisper and Text-to-Speech via Azure & Eleven Labs. Run locally in your browser with no need to install applications, featuring faster performance than the official ChatGPT UI by connecting directly to the API. Perfect for users who want privacy, speed, and voice interaction with their AI assistant.
