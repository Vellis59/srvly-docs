# Figrarium

> Figrarium est un outil d'orchestration de pipelines data avec une approche visuelle. Il s'adresse aux data engineers et analystes qui veulent créer des pipelines de transformation (ETL) sans tout coder en Python ou SQL. ⚠️ Note : projet peu documenté publiquement — fiche de prise en main, à enrichir
