# Teemii

> Teemii est une application web taillée pour les amateurs de manga, comics et webtoons qui veulent une expérience unifiée de lecture dans le navigateur. Le projet se positionne à la croisée de Kavita (gestionnaire de bibliothèque) et de HakuNeko (lecteur/scraper) : Teemii propose à la fois la gestion
