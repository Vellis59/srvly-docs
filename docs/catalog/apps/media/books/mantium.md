# Mantium

> ⚠️ Note honnête : Mantium est catégorisé par selfh.st sous le tag Books, mais son objet réel est le suivi de mangas (et plus largement comics / light novels / manhwas), pas la lecture de livres. Il ne télécharge aucun chapitre : il agrège les métadonnées (titre, couverture, chapitres, dates de sorti
