# ComicOPDS

> ComicOPDS est un mini-serveur OPDS écrit en Python / Flask dont la fonction est on ne peut plus simple : scanner un dossier rempli de CBR/CBZ et exposer un flux OPDS 1.2 que n'importe quelle liseuse compatible sait consommer. Pas de base de données à monter, pas d'interface web : juste un endpoint H
