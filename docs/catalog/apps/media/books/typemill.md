# Typemill

> Typemill est un CMS flat-file basé sur Markdown, conçu pour les sites éditoriaux : manuels utilisateur, documentation technique, knowledge bases, wikis, handbooks, livres numériques. Pas de base de données : tout est stocké en fichiers Markdown dans `content/`, ce qui rend les sauvegardes triviales
