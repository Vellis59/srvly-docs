# BiblioReads

> BiblioReads est un lecteur EPUB en ligne radicalement simple : vous déposez un fichier `.epub` dans l'interface, et il s'ouvre dans le navigateur avec un rendu soigné (pagination, marque-pages, progression). Pas de base de données centrale, pas de comptes, pas de synchronisation cloud : la philosoph
