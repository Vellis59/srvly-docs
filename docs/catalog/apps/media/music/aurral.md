# Aurral

> Aurral is a self-hosted music discovery and request management app for Lidarr. Search artists via MusicBrainz, add them to Lidarr with granular monitoring options, browse your library, and discover new artists from recommendations. Features include Weekly Flow playlists (powered by Soulseek + Navidrome), multi-user support with permissions, Gotify notifications, and PWA support. Aurral is designed to be safe for your collection—library changes go through Lidarr's API.
