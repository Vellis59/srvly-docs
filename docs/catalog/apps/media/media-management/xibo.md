# Xibo

> Xibo est une plateforme complète d'affichage dynamique : CMS web pour concevoir et planifier des layouts (images, vidéos, web, flux RSS, données), un serveur central, et des players (Xibo Player, Android, web) qui se connectent pour afficher le contenu sur écrans.
