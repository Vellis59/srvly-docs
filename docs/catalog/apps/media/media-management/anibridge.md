# AniBridge

> AniBridge is a media synchronization tool that keeps your anime watch activity perfectly in sync across library providers (Plex, Jellyfin, Emby) and list trackers (AniList, MyAnimeList). It includes a web dashboard with a real-time sync timeline, profile manager, custom mapping editor, log viewer, and automatic list backups.
