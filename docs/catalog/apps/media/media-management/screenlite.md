# Screenlite

> Screenlite est une plateforme d'affichage dynamique moderne 100% web (CMS + player), avec multi-tenancy, layouts configurables (images, vidéos, web, embeds), planification horaire, et remote management. Différence vs [[app-xibo]]: Screenlite est plus récent, plus simple, full TypeScript moderne, san
