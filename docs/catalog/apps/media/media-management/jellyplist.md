# Jellyplist

> Jellyplist était un « Spotify → Jellyfin bridge » : vous ajoutez une playlist Spotify dans Jellyplist, l'app la synchronise dans votre instance [[app-jellyfin]] en téléchargement les morceaux manquants via spotDL et en créant/met à jour une playlist m3u native côté Jellyfin.
