# Neko

> Neko est un "browser-in-browser" auto-hébergé : il lance une instance de Chromium, Firefox ou VLC dans un conteneur Docker et la streame vers les navigateurs des participants, avec contrôle souris/clavier partagé. Cas d'usage phares : soirée anime à distance, coworking sur un même document, démo cli
