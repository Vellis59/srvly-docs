# Scrypted

> Scrypted est une plateforme NVR open source moderne qui se positionne comme un hub domotique vidéo : il ingère des flux RTSP/ONVIF de caméras IP, les ré-encode (H.264, H.265, MJPEG) et les expose à HomeKit, Google Home, Alexa, Home Assistant via des plugins officiels.
