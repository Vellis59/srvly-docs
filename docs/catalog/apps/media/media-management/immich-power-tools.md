# Immich Power Tools

> Immich Power Tools est né d'un constat simple : Immich est un gestionnaire de photos excellent, mais certaines opérations de masse (renommer des centaines de photos, appliquer un tag à un album entier, nettoyer les doublons) restent fastidieuses dans son UI officielle.
