# MediaCMS

> MediaCMS est une plateforme complète de partage/streaming vidéo avec upload, transcodage HLS, lecteurs HTML5, playlists, commentaires, et modules payants. Différence vs PeerTube : architecture centralisée (pas de P2P), fonctionnalités e-commerce intégrées. Pour qui : médias, associations, formateurs
