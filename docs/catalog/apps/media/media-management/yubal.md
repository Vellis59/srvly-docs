# yubal

> yubal est un projet récent (fin 2025) qui se concentre sur un créneau précis : télécharger depuis YouTube Music avec un taggage propre, prêt à importer dans un lecteur. Là où [[app-yt-dlp-web-player]] reste généraliste (vidéo, audio, formats), yubal va plus loin sur la partie *musique* : récupératio
