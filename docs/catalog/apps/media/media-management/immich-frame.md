# Immich Frame

> Immich Frame est un projet officiel du team Immich (sous l'organisation GitHub `immich-app`) qui matérialise un usage simple : afficher votre bibliothèque Immich en mode cadre photo numérique. Vous avez un vieil écran, une tablette Android murale, un Pi avec écran LCD ? Branchez Immich Frame et vous
