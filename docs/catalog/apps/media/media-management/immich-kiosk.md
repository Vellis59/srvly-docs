# Immich Kiosk

> Immich Kiosk est un projet communautaire (par `damongolding`) qui se branche sur votre instance Immich pour offrir un mode présentation automatique : pas d'UI, pas de menus, juste vos photos qui défilent en plein écran, avec des transitions soignées.
