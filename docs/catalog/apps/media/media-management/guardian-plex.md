# Guardian (Plex)

> Guardian est un daemon simple qui surveille l'état d'un serveur [[app-plex]] et déclenche des alertes via Discord, Telegram, [[app-ntfy]], email, ou webhook générique. Contrairement à [[app-tautulli]] qui est un dashboard complet (sessions actives, stats, etc.), Guardian reste focalisé sur l'essenti
