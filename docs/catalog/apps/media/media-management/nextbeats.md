# NextBeats

> NextBeats n'est pas un *music server* à proprement parler — c'est un lecteur ambiant web dans la veine de Lofi Girl ou Radio Garden, mais self-hostable. L'idée : reproduire l'expérience d'un stream lofi pour coder/étudier, avec la possibilité d'empiler des couches d'ambiance (pluie, café, feu, vent)
