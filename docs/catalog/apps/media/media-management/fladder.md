# Fladder

> Fladder est un client alternatif pour Jellyfin développé en Flutter, ce qui lui permet de tourner nativement sur Linux, Windows, macOS, Android, iOS et le web à partir d'une seule base de code. Il se positionne comme une alternative moderne au client web officiel de Jellyfin, plus rapide et plus est
