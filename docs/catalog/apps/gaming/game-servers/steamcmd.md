# SteamCMD

> SteamCMD is a command-line version of the Steam client for installing and updating Steam games and dedicated servers. It's perfect for managing game servers and game installations.

## Quick Info

| Field | Value |
|-------|-------|
| **Category** | Gaming |
| **Subcategory** | Game Servers |
| **Default Port** | 80 |
| **Docker Image** | `steamcmd/steamcmd:latest` |
| **Dependencies** | Docker |
| **OS Support** | Ubuntu 22.04, Ubuntu 24.04, Debian 12 |
