# auto-mcs

> auto-mcs (prononcer "auto-mc-s") est un gestionnaire de serveurs Minecraft pensé pour les joueurs occasionnels qui veulent monter un serveur multijoueur pour leurs amis sans se prendre la tête. La promesse : un serveur prêt en moins d'une minute, sans configurer Java, sans chercher le bon modloader,
