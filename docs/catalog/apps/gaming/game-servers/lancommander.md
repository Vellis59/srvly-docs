# LANCommander

> LANCommander est né d'un besoin simple : organiser des LAN parties sans que chaque participant doive télécharger 50 Go de jeux sur clé USB, et sans dépendre d'Internet (fréquent dans les salles pas forcément connectées). L'idée est un Steam-like privé : un serveur central qui héberge la bibliothèque
