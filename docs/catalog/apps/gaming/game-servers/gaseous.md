# Gaseous

> Gaseous est un gestionnaire de ROMs self-hosted qui combine trois fonctions en une seule application : identification automatique des jeux par hash de fichier, récupération de métadonnées depuis plusieurs sources (IGDB, Hasheous), et émulation en navigateur via EmulatorJS.
