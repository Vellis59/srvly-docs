# Foundry Virtual Tabletop

> Foundry Virtual Tabletop (souvent abrégé Foundry VTT ou FVTT) est une application web de Virtual Tabletop pensée pour les maîtres de jeu de rôle (D&D 5e, Pathfinder, Call of Cthulhu, Shadowrun, etc.). Contrairement à ses concurrents historiques, Foundry se distingue par son modèle one-time-purchase
