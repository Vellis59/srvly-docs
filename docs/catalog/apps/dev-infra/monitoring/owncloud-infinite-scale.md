# ownCloud Infinite Scale

> ownCloud Infinite Scale (abrégé oCIS) est né en 2020 sous l'impulsion de l'équipe d'ownCloud GmbH pour répondre à deux limitations structurelles du PHP historique : les goulets d'étranglement en I/O (PHP-FPM ne tient pas la charge de millions de fichiers) et l'impossibilité d'exposer une vraie API S
