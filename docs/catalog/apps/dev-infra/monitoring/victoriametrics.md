# VictoriaMetrics

> VictoriaMetrics est une base de données time-series (TSDB) conçue comme un drop-in replacement de Prometheus, avec des performances nettement supérieures en termes de : - Vitesse d'ingestion : jusqu'à 20× plus rapide que Prometheus - Consommation RAM : 10-70× moins de RAM - Compression disque : 5-10
