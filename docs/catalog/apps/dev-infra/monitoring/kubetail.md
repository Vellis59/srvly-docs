# Kubetail

> Kubetail est un dashboard de logs Kubernetes qui résout le plus gros point de friction de `kubectl` : suivre les logs de plusieurs pods simultanément. Fini le jonglage entre 5 terminaux avec `kubectl logs -f pod-1`, `kubectl logs -f pod-2`, etc. Kubetail unifie tout dans une seule UI web (ou TUI) av
