# OneUptime

> OneUptime est la plateforme d'observabilité open source la plus ambitieuse : elle ambitionne de remplacer à elle seule Datadog (monitoring) + PagerDuty (on-call) + Statuspage (status publique) + Better Uptime (synthetics), le tout en une seule application self-hosted. Issue d'une société commerciale
