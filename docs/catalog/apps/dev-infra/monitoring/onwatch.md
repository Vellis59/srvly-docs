# onWatch

> onWatch est une plateforme de veille personnalisable qui combine des sources de données hétérogènes (API crypto, sites e-commerce, flux RSS, réseaux sociaux, indices boursiers) dans une interface unifiée de watchlist. Contrairement aux agrégateurs classiques, onWatch intègre un agent IA qui apprend
