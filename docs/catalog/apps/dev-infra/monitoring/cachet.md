# Cachet

> Meet Cachet, the open-source status page system designed to improve downtime communication. Used by great companies worldwide, Cachet allows you to effectively communicate downtime and system outages to your customers, teams, and shareholders. It comes with a beautifully crafted design that is responsive and works seamlessly on your desktop, tablet, and phone. Cachet supports multiple languages and comes with a powerful JSON API for updating your status page with incidents and component status changes. It also allows you to schedule maintenance events ahead of time and provides a dashboard for setting up metrics to measure various aspects such as uptime, error rates, etc. For added security, Cachet supports two-factor authentication. It's a beautifully simple service for managing your service statuses.
