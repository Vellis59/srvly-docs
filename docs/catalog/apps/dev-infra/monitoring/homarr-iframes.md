# Homarr iFrames

> À l'origine, Homarr ne supportait qu'un nombre limité d'intégrations natives (Sonarr, Radarr, Plex…). Pour les services sans connecteur officiel, il fallait ruser avec des iframes brutes, ce qui donnait un rendu moyen. Homarr iFrames comble ce manque : un module complémentaire qui ajoute une tuile i
