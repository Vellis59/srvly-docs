# Fail2Ban-Report

> Fail2Ban-Report est un dashboard web léger spécialement conçu pour visualiser les données de Fail2Ban, le célèbre outil de protection contre le brute-force (SSH, FTP, HTTP, SMTP, etc.). Fail2Ban génère énormément de logs utiles (qui est banni, depuis quand, pour quel service, combien de tentatives)
