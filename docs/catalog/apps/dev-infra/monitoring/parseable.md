# Parseable

> Parseable est un système d'analytics de logs moderne écrit en Rust qui stocke les événements au format Parquet (columnar) directement sur du stockage objet (S3, MinIO, filesystem). C'est la même approche que les data warehouses modernes (Snowflake, ClickHouse, DuckDB) appliquée aux logs : compressio
