# Logdy

> Logdy est un viewer de logs en temps réel qui transforme n'importe quel flux de logs (stdin, fichiers, syslog, Docker) en une interface web moderne accessible depuis le navigateur. C'est le successeur moderne du bon vieux `tail -f | grep` : même philosophie (simplicité), mais avec une UI qui permet
