# Pulse

> Pulse est un tableau de bord moderne pour Proxmox VE qui se greffe sur l'API REST du cluster pour afficher en temps réel l'état des nœuds, des VM et des containers LXC. Pensé par et pour les utilisateurs de Proxmox, c'est la pièce qui manque à l'écosystème : l'interface web de Proxmox est puissante
