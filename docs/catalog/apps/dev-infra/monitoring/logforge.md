# LogForge

> LogForge est un dashboard de monitoring léger pour services Docker qui combine visualisation de logs, status de santé, métriques de ressources (CPU/RAM/disk) et actions de cycle de vie (start/stop/restart) dans une seule UI. C'est l'outil idéal pour les petits/moyens labos qui veulent gérer visuelle
