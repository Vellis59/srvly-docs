# Nimtable

> Nimtable est une plateforme web libre qui se place entre les utilisateurs et les catalogues Apache Iceberg pour offrir une console de gestion et une API REST Iceberg normalisée. Elle permet d'explorer les schémas, partitions, snapshots et manifestes d'un data lake, d'exécuter des requêtes SQL direct
