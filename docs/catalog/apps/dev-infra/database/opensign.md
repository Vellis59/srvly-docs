# OpenSign

> OpenSign est une solution de signature électronique open source positionnée pour les PME et ETI qui veulent intégrer la signature dans leur workflow existant. Le projet se distingue par son mode "embed" : OpenSign peut être embarqué dans une autre application via iframe/API, permettant aux entrepris
