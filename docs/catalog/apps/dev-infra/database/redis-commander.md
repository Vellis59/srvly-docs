# Redis Commander

> Redis Commander est une application web Node.js qui propose une interface graphique complète pour administrer des serveurs Redis (jusqu'à Redis 7+ et Redis Stack). Elle permet de visualiser, ajouter, modifier et supprimer des clés de tous types (string, hash, list, set, zset, stream, JSON) sans avoi
