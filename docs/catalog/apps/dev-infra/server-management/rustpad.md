# Rustpad

> Rustpad est un éditeur de code collaboratif temps réel (basé sur Operational Transform / CRDT) avec coloration syntaxique pour 100+ langages, sans tracking ni télémétrie. Les pads sont chiffrés en transit (HTTPS) et le serveur n'a pas besoin de persister le contenu. Différence vs VS Code Live Share
