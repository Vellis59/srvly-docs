# Flint

> Flint est un outil de gestion de configuration (provisioning) pour serveurs Linux. Il vise à remplacer Ansible avec un DSL plus simple, écrit en Rust pour des performances élevées. On déclare l'état souhaité du serveur dans un fichier `flint.toml` et l'app s'occupe d'exécuter les commandes nécessair
