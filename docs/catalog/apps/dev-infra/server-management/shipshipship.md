# ShipShipShip

> ShipShipShip est un clone self-hosted de Product Hunt (et de ses alternatives comme BetaList, Launching Next) : il permet de créer des pages de lancement pour vos produits, avec un système de vote, classement, commentaires, badges (« #1 Product of the Day ») et une file d'attente pour les lancements
