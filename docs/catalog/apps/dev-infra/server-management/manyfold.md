# Manyfold

> Manyfold est un gestionnaire de modèles 3D pour votre collection locale (STL, OBJ, 3MF, GLB). Il scanne un dossier, indexe les fichiers, génère des aperçus 3D, gère les métadonnées, tags, créateurs. Pensé pour les makers qui accumulent des centaines de fichiers et veulent s'y retrouver sans dépendre
