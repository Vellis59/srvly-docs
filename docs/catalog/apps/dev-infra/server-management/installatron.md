# Installatron

> Installatron est un auto-installer commercial pour applications web qui équipe historiquement des milliers d'hébergeurs mutualisés (cPanel, Plesk, DirectAdmin, InterWorx) à travers le monde. Sa mission : permettre à un utilisateur final non-technique d'installer, mettre à jour, sauvegarder, restaure
