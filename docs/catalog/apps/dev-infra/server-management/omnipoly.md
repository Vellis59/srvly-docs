# OmniPoly

> OmniPoly est un front-end alternatif pour Poly Haven, la célèbre bibliothèque d'assets 3D, textures et HDRIs 100% libres (CC0) utilisée massivement par les graphistes, architectes, développeurs de jeux et makers Blender. Le site officiel est très bien fait, mais OmniPoly ajoute une couche self-hosta
