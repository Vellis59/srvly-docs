# Jellyswarrm

> Jellyswarrm est un reverse proxy L7 ultra-spécialisé : il se place devant plusieurs instances Jellyfin et les présente aux clients (Jellyfin Mobile, web, TV, etc.) comme un seul serveur unifié. C'est la réponse au problème classique : "j'ai un serveur Jellyfin chez moi, mon ami en a un autre à 500 k
