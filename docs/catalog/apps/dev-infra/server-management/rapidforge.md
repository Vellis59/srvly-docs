# RapidForge

> RapidForge est une plateforme d'hébergement de builds et d'intégration continue open source, pensée comme une alternative auto-hébergeable aux services SaaS type Travis CI, CircleCI ou AppVeyor. Le projet se positionne sur la simplicité de déploiement et l'absence de vendor lock-in : un binaire uniq
