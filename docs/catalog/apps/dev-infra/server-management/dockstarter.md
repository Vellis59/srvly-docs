# DockSTARTer

> DockSTARTer est un ensemble de scripts Bash qui automatisent l'installation et la configuration d'une stack Docker complète sur une machine Linux nue (typiquement Debian, Ubuntu, ou Raspberry Pi OS). L'objectif est de fournir un équivalent Linux de "Synology" pour les bricoleurs : on installe l'OS,
