# Rundeck

> Rundeck est l'outil de référence pour l'exécution sécurisée de scripts sur des flottes de serveurs hétérogènes, avec une UI web, des logs centralisés, du RBAC, et un scheduler cron. Là où [[app-ansible|Ansible]] est opinionated (YAML, SSH, modules), Rundeck est agnostique : un Job Rundeck peut lance
