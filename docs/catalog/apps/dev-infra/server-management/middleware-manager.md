# Middleware Manager

> Middleware Manager (par HHF Technology, mêmes éditeurs que Traefik Log Dashboard) est un microservice qui génère la configuration dynamique de Traefik à partir d'une interface web. Il résout la douleur classique de Traefik : les middlewares sont puissants mais syntaxiquement pénibles à écrire en YAM
