# Sink

> Sink est un raccourcisseur d'URL moderne écrit en Rust, initialement conçu pour être déployé sur Cloudflare Workers et Cloudflare Pages (KV + D1) pour des performances edge. Il fonctionne également en mode Node.js classique derrière un reverse proxy pour les utilisateurs qui préfèrent Docker et un V
