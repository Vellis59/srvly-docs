# MicroBin

> MicroBin est un micro-service de partage de texte et de fichiers ultra-léger, conçu pour être déployé en 30 secondes sur un petit VPS ou un Raspberry Pi. Il gère à la fois les pastes (texte, code, markdown), les fichiers (jusqu'à plusieurs Go), les URLs courtes (redirection) et même un mini file bro
