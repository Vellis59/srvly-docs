# Stormkit

> Stormkit est un PaaS (Platform-as-a-Service) spécialisé dans le déploiement d'apps frontend : c'est le « Vercel/Netlify self-hosted » que beaucoup attendent. Vous connectez un repo GitHub/GitLab/Bitbucket, vous choisissez le framework (React, Vue, Svelte, Solid, Astro, Next, Nuxt, SvelteKit, Gatsby…
