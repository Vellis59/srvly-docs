# Diun

> Diun (Docker Image Update Notifier) est un outil écrit en Go qui scanne périodiquement les images Docker que vous utilisez (via votre `docker-compose.yml` ou ses propres labels) et vous notifie quand une nouvelle version est disponible sur le registry (Docker Hub, GHCR, Quay, ECR, registries privés…
