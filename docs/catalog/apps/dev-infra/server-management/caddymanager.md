# CaddyManager

> CaddyManager est une WebUI construite sur la stack MEVN (MongoDB / Express / Vue / Node) qui permet de gérer à distance plusieurs serveurs Caddy depuis une interface unique. Le projet répond à un reproche fréquent fait à [[app-caddy]] : sa configuration via `Caddyfile` est simple, mais pas d'interfa
