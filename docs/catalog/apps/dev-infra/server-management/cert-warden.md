# Cert Warden

> Cert Warden (anciennement *Cert Warden* par gregtwallace) est un client ACME centralisé : au lieu que chaque service (reverse proxy, NAS, serveur mail, etc.) gère ses propres certificats Let's Encrypt indépendamment, un seul point les génère, les stocke, et les sert via une API authentifiée par clés
