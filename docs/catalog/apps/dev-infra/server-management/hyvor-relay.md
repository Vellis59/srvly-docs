# Hyvor Relay

> Hyvor Relay est un service de mailing transactionnel self-hosted qui permet d'envoyer des emails applicatifs (confirmations, magic links, notifications, receipts, OTP) via SMTP (port 1025) ou API HTTP REST. C'est la réponse open source à Postmark, SendGrid et Amazon SES : contrôle total des données,
