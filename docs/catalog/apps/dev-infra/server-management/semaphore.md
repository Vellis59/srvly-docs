# Semaphore

> Semaphore (nom complet *Ansible Semaphore*) est une interface web moderne pour Ansible qui répond au besoin classique : "je veux que l'équipe puisse lancer les playbooks prod depuis une UI, avec logs, secrets, et audit, sans donner accès à la CLI". C'est l'anti-AWX : là où AWX est une montagne Kuber
