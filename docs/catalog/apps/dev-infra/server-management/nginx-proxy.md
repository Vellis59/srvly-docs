# nginx-proxy

> nginx-proxy est le pionnier du reverse proxy "Docker-native" : un conteneur unique qui wrap Nginx + docker-gen (générateur de configs basé sur les métadonnées Docker). Quand un conteneur démarre avec `VIRTUAL_HOST=app.example.com`, nginx-proxy détecte l'événement, génère la conf Nginx, et reload — z
