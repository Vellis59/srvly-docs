# Mail Archiver

> Mail Archiver est une application open source dédiée à l'archivage d'e-mails : elle permet d'importer des boîtes entières (formats mbox, Maildir, PST/EML), de les indexer en full-text, de les rechercher rapidement via une interface web, et de les stocker dans un format pérenne (SQLite ou Postgres +
