# Bichon

> Bichon est un outil open source d'archivage de flux RSS/Atom : il récupère régulièrement vos abonnements, conserve l'historique complet des articles (pas seulement les derniers), les indexe en full-text et les expose via une interface web minimaliste. Pensé pour ceux qui veulent garder une trace dur
