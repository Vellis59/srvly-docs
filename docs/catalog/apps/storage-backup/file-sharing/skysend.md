# SkySend

> SkySend est un service de partage de fichiers relativement récent qui se positionne sur le créneau des solutions "user-friendly" pour envoyer des fichiers volumineux à des destinataires externes. L'accent est mis sur l'expérience utilisateur : upload fluide, design soigné, prévisualisations automati
