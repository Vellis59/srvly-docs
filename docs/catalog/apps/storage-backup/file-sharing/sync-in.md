# Sync-in

> Sync-in est un projet plus modeste que [[app-nextcloud]] ou [[app-cloudreve]], mais qui remplit efficacement sa mission : offrir un cloud storage personnel simple à déployer pour un particulier ou une petite équipe, sans les lourdeurs d'une suite collaborative complète. Le choix de Node.js + Express
