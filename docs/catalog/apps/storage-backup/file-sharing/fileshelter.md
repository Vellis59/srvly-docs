# FileShelter

> A simple and secure file sharing application that allows users to upload and share files with a clean web interface. Files are stored on the server and can be accessed through unique URLs.

## Quick Info

| Field | Value |
|-------|-------|
| **Category** | Storage & Backup |
| **Subcategory** | File Sharing & Sync |
| **Default Port** | 5091 |
| **Docker Image** | `epoupon/fileshelter:6.4.0` |
| **Dependencies** | Docker |
| **OS Support** | Ubuntu 22.04, Ubuntu 24.04, Debian 12 |
