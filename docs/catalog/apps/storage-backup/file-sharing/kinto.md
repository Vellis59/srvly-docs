# Kinto

> Kinto est un projet de la fondation Mozilla (initialement par Mozilla Services puis piloté par la communauté) qui propose un magasin de données JSON exposé via une API REST simple, conçu pour être répliqué et synchronisé entre plusieurs clients ou instances. L'idée est proche de CouchDB ou Firebase,
