# Transfer.zip

> Transfer.zip est un clone open source de WeTransfer conçu pour être self-hosté. L'objectif est de fournir une expérience utilisateur familière (interface similaire à WeTransfer) tout en gardant le contrôle des données. Les fichiers sont uploadés via une interface web moderne, avec liens partageables
