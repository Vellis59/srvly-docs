# Git-Sync

> Git-Sync est un service léger qui clone et met à jour périodiquement une liste de dépôts Git vers un dossier local ou un volume....

## Quick Info

| Field | Value |
|-------|-------|
| **Category** | Storage & Backup |
| **Subcategory** | Backup & Restore |
| **Dependencies** | Docker |
| **OS Support** | Ubuntu 22.04, Ubuntu 24.04, Debian 12 |
