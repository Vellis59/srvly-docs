# Unraid Guardian

> Unraid Guardian est un script shell qui transforme la configuration d'un serveur Unraid en une archive portable, chiffrée et versionnée, permettant de tout reconstruire à l'identique sur une clé USB neuve ou un nouveau serveur Unraid. C'est l'outil de disaster recovery spécifique à l'écosystème Unra
