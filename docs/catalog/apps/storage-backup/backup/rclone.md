# Rclone

> Rclone est l'outil de référence pour synchroniser, monter, copier et chiffrer des fichiers entre une machine locale et plus de 70 services cloud : S3, Backblaze B2, Google Drive, Dropbox, OneDrive, Azure Blob, SFTP, WebDAV, Nextcloud, Google Cloud Storage, etc. Il se manipule exclusivement en ligne
