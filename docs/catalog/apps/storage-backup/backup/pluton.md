# Pluton

> Pluton se positionne comme une solution de backup unifiée : un serveur central de coordination + des agents (clients) sur chaque machine à sauvegarder. Contrairement à [[app-restic]] ou [[app-borg]] qui sont des outils « un client = un repository », Pluton orchestre tout depuis une seule interface,
