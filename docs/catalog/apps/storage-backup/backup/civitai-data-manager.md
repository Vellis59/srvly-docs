# Civitai Data Manager

> Civitai Data Manager ne sauvegarde pas les modèles eux-mêmes (un checkpoint LoRA fait couramment 2 à 7 Go, et certains packs pèsent des dizaines de Go). Il se concentre sur ce qui est petit mais critique : les métadonnées Civitai associées à chaque modèle, que vous perdez si Civitai modifie ou suppr

## Quick Info

| Field | Value |
|-------|-------|
| **Category** | Storage & Backup |
| **Subcategory** | Backup & Restore |
| **Dependencies** | Docker |
| **OS Support** | Ubuntu 22.04, Ubuntu 24.04, Debian 12 |
