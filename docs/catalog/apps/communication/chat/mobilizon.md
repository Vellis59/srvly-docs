# Mobilizon

> Mobilizon est le projet de plateforme d'événements fédérée porté initialement par Framasoft, l'association française qui a popularisé le slogan "dégooglisons Internet". L'objectif est limpide : offrir un équivalent de Meetup, Eventbrite ou Facebook Events mais décentralisé, sans traque publicitaire,
