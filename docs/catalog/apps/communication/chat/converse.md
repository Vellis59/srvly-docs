# Converse.js

> Converse.js est un client XMPP web complet, écrit en JavaScript, conçu pour fonctionner à la fois comme widget intégrable dans une page existante, comme PWA installable, ou comme client standalone auto-hébergé. Ce n'est pas un serveur : il se connecte à n'importe quel serveur XMPP (Prosody, ejabberd
