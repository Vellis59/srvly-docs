# SimpleLogin

> SimpleLogin a été créé en 2019 par Son Tran (ingénieur Notion) puis acquis par Proton en 2022. Le principe : créer un alias unique par service (ex. `amazon-xyz@maison.sl` pour Amazon) qui forwarde vers votre vraie boîte mail. Si un alias commence à recevoir du spam, vous le désactivez → fini le spam
