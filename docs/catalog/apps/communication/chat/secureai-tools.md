# SecureAI Tools

> SecureAI Tools est une plateforme d'IA « clé en main » pensée pour les particuliers, familles et petites équipes qui veulent bénéficier d'un assistant conversationnel et d'un moteur de RAG (Retrieval-Augmented Generation) sans confier leurs conversations ni leurs documents à un service tiers.
