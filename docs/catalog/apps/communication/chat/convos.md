# Convos

> Convos est né d'une frustration : IRC est un protocole excellent et fédéré, mais les clients historiques (irssi, weechat, HexChat) demandent de l'investissement pour être utilisés confortablement au quotidien. Convos propose une interface web moderne, persistante et multi-appareils pour IRC et IRC-o
