# Wiki-Go

> Wiki-Go (par *LeoMoon Studios*) est un wiki flat-file écrit en Go qui se positionne sur le créneau *"No database. No bloat. Zero maintenance. Just Markdown."*. Concrètement, il n'y a aucune base de données à configurer : tout est stocké comme fichiers Markdown sur disque, ce qui rend l'installation
