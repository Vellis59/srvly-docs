# LeafWiki

> LeafWiki est un wiki self-hosted qui assume pleinement la philosophie *"une documentation long-terme doit être lisible en dehors de l'application"*. Concrètement : le contenu des pages est stocké comme de simples fichiers `.md` sur disque, et seule la métadonnée (utilisateurs, tags, index de recherc
