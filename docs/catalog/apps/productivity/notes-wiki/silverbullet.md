# SilverBullet

> SilverBullet se décrit comme une base de connaissances personnelle privée, auto-hébergeable et programmable, construite autour de fichiers Markdown. C’est une solution très attractive pour les utilisateurs qui aiment les liens wiki, les backlinks, les objets structurés et surtout l’idée de faire évo
