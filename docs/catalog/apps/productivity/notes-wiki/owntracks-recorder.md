# OwnTracks Recorder

> OwnTracks est un écosystème complet de géolocalisation : - OwnTracks Mobile : app iOS/Android qui publie la position - OwnTracks Recorder : serveur qui reçoit et stocke - OwnTracks Frontend : interface web cartographique
