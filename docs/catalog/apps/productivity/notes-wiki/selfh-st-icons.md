# selfh.st/icons

> selfh.st/icons est une collection communautaire d'icônes SVG/PNG pour plus de 2000 applications self-hosted, à utiliser dans vos dashboards, README, sites web. Compatible DashboardIcons, Heimdall, Homarr, etc. Différence vs Iconify/Flaticon : focus self-hosted, qualité cohérente, noms uniformes, CDN
