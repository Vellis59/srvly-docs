# Plex Rewind

> Plex Rewind se connecte à votre serveur Plex, lit l'historique Tautulli/Plex et génère un récap annuel visuel (cartes partageables type Wrapped) : top 5 films, séries, genres, heures totales, statistiques par mois. Différence vs Spotify Wrapped : données 100% locales, configurable, partage libre. Po
