# Kanba

> Kanba est un Kanban open source et minimaliste conçu pour les développeurs qui veulent un clone Trello simple, rapide et moderne, sans la lourdeur d'un [[app-wekan]] ou la complexité d'un [[app-openproject]]. La stack est volontairement classique : React (Create React App) pour le front, Node.js/Exp
