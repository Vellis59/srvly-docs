# Overleaf

> Overleaf Community Edition est la version self-hostable de l'éditeur LaTeX collaboratif le plus utilisé au monde (utilisé par Springer, Nature, universités). Édition WYSIWYG + LaTeX en temps réel, compilation cloud via TeX Live, Git integration, historique de versions. Différence vs TeXmaker/Texstud
