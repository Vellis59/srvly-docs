# Davis

> Davis est un serveur CalDAV/CardDAV moderne (PHP/Symfony) avec UI web intégrée, partage de calendriers, support multi-utilisateurs, LDAP, deux facteurs. Compatible Apple iOS/macOS, Mozilla Thunderbird, DAVx5 Android. Différence vs Radicale/Baikal : UI web moderne, Symfony maintenable, support CardDA
