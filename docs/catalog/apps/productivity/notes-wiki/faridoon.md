# Faridoon

> Faridoon est un suivi de prix minimaliste : on ajoute l'URL d'une fiche produit (Amazon, Cdiscount, Fnac, sites persos…), on indique le prix cible, et l'application scrape périodiquement la page pour historiser l'évolution.
