# YouLag

> YouLag combine deux fonctions : (1) un proxy/cache YouTube qui pré-charge les vidéos populaires pour les servir rapidement depuis le réseau local, et (2) une génération de flux RSS pour chaînes/playlists YouTube (alternative à Invidious pour la veille RSS).
