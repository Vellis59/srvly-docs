# Workout Tracker

> Workout Tracker est une application web légère qui fait exactement ce qu'on attend d'un tracker d'entraînements : définir des exercices, créer des routines, logger les séries (poids, répétitions), et visualiser la progression dans le temps. Pas de fonctionnalités gadget, pas de nutrition, pas de rés
