# Pasta

> Pasta est un service léger pour gérer, normaliser et synchroniser les sous-titres d'une bibliothèque vidéo (films, séries) en se basant sur les hashs OpenSubtitles et le hash MediaInfo des fichiers. Différence vs Bazarr : interface plus minimaliste, focus recherche/sync. Pour qui : utilisateurs de P
