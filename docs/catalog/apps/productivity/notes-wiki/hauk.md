# Hauk

> Hauk est un backend open source de partage de localisation : vous créez un lien, vous le partagez à vos contacts, ils voient votre position se mettre à jour sur une carte OpenStreetMap en temps réel, jusqu'à expiration configurable. Mode "receveur" pour voir la position d'un autre membre. Pas d'app,
