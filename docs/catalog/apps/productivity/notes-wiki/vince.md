# Vince

> Vince est un outil d'analyse web minimaliste écrit en Go qui rejoint la famille des analytics "à la GoatCounter" : single binary, SQLite par défaut, zéro cookie, script ultra-léger et déploiement en 30 secondes. La philosophie est claire : fournir l'essentiel des métriques web (visiteurs, pages vues
