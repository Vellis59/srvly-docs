# Relaticle

> Relaticle est un CRM jeune construit sur Laravel 11 + Filament 3, positionné sur la gestion relationnelle d'équipes B2B : contacts, sociétés, opportunités, tâches, notes. Cible affichée : remplacer Notion/HubSpot pour les fondateurs de startups.
