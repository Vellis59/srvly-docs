# Atuin

> Atuin remplace le traditionnel `~/.bash_history` par une base SQLite locale, liée à un serveur central optionnel qui synchronise l'historique entre toutes les machines d'un même utilisateur. La fonction première est de retrouver une commande exécutée il y a six mois grâce à une recherche fuzzy ultra
