# Forms.md

> Forms.md est une approche minimaliste : vous écrivez votre formulaire dans un fichier `.md` (front-matter + champs YAML), le serveur Go le transforme en page HTML servie via une URL publique. Les réponses arrivent par email ou via webhook.
