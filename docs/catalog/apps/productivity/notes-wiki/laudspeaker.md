# Laudspeaker

> Laudspeaker est une alternative open source à Customer.io / Braze : gestion de segments, automatisations multi-étapes, templates email/push/in-app, tracking d'événements utilisateur, A/B testing. Différence vs Customer.io : 100% auto-hébergeable, données jamais partagées. Pour qui : SaaS en croissan
