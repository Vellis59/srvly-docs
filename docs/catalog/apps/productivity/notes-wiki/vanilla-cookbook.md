# Vanilla Cookbook

> Vanilla Cookbook est un gestionnaire de recettes minimaliste écrit en PHP, pensé pour les utilisateurs qui veulent un outil simple et fonctionnel sans la complexité des gros projets comme Mealie ou Tandoor. Pas de framework JS moderne, pas de base PostgreSQL à administrer : juste du PHP, un peu de J
