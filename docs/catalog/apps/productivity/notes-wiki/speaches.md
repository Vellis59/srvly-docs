# Speaches

> Speaches (jeu de mots sur *speech* + *es*) est un serveur HTTP compatible avec l'API OpenAI Audio (`/v1/audio/transcriptions`, `/v1/audio/translations`, `/v1/audio/speech`) qui s'installe en une ligne de Docker et accepte n'importe quel client OpenAI existant : Home Assistant, Open WebUI, Cody, Cont
