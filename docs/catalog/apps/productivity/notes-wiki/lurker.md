# Lurker

> Lurker est un front-end Reddit conçu spécifiquement pour les utilisateurs qui ne font que lire (les "lurkers"), c'est-à-dire qui n'ont ni besoin ni envie de se connecter pour consommer du contenu. Écrit en Go pour offrir des performances solides avec une empreinte mémoire minimale, il sert de proxy
