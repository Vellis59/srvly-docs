# World Monitor

> World Monitor est un projet d'envergure qui agrège en continu des sources ouvertes (OSINT) pour les afficher sur un tableau de bord cartographique unique. On y retrouve les zones de conflit, les séismes, les trajectoires d'avions et de navires, les variations de marchés financiers, etc.
