# Puter

> Puter est un système d'exploitation web offrant un bureau complet accessible depuis le navigateur : fichiers, apps, terminal, multi-fenêtres, comptes utilisateurs avec stockage. Différence vs Nextcloud : expérience desktop traditionnelle, apps tierces, GUI façon macOS/Linux. Pour qui : utilisateurs
