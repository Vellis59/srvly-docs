# webtrees

> webtrees est une application web de généalogie collaborative qui lit les fichiers GEDCOM (format standard), affiche les arbres familiaux interactifs, les fiches individus, relations, cartes (intégration OpenStreetMap), et accepte plusieurs utilisateurs/clans.
