# Sprout Track

> Sprout Track est né d'un besoin très concret : un jeune papa a développé l'application pour sa propre femme afin de suivre les activités du nouveau-né au quotidien. Cette origine « parent-first » se ressent dans l'ergonomie : l'interface est mobile-first, optimisée pour être utilisée d'une seule mai
