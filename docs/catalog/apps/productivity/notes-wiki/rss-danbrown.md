# RSS

> Ce projet, hébergé sur Codeberg par l'utilisateur `danb`, est un agrégateur RSS d'une grande simplicité : tous les articles de tous les feeds abonnés s'affichent en un fil chronologique unique, façon timeline. L'interface est dépouillée, sans catégories, sans filtres, sans dossiers.
