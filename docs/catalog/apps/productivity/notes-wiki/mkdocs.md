# MkDocs

> MkDocs est un générateur de site statique spécialisé documentation, écrit en Python, qui transforme un dossier de fichiers Markdown en un site web statique propre et navigable. Sa philosophie : « project documentation with Markdown ». L'outil est volontairement minimaliste, sans JS lourd, sans base
