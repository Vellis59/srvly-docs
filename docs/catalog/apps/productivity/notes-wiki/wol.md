# Wake-on-LAN (Wol)

> Wol (ou Shortwave) est un serveur Wake-on-LAN minimaliste écrit en Go, fournissant une API REST + UI web pour réveiller des machines à distance via magic packet. Supporte broadcast direct, listes d'hôtes favorites, scheduler (cron-like), multi-interfaces réseau. Différence vs etherwake CLI : API/UI
