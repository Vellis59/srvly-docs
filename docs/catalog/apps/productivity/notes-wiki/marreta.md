# Marreta

> Marreta est un proxy léger inspiré de 13ft Ladder, conçu pour la presse brésilienne (Folha, Estadão, Globo) mais fonctionnel sur la plupart des sites paywall. Il nettoie les cookies, retire les overlays de paywall, et renvoie l'article complet. Différence vs 13ft Ladder : optimisé pour les sites BR,
