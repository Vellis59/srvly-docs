# Drivebase

> Drivebase est un projet récent qui répond à une frustration courante des utilisateurs intensifs de Google Drive : l'interface web officielle est lente, l'organisation par dossiers est rigide, et le multi-comptes est pénible. Drivebase propose une UI alternative, plus rapide, avec support de plusieur
