# Subtrackr

> Subtrackr est un tracker d'abonnements multi-utilisateurs (familles, équipes, colocataires) avec partage des dépenses, division de facture (split), API REST et webhooks. Différence vs Wallos/Wapy: Subtrackr pousse le collaboratif (qui paie quoi, qui rembourse). Pour qui: familles, couples, colocatai
