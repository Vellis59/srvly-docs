# MarpUI

> MarpUI offre une interface graphique web au moteur Marp (Markdown Presentation Ecosystem), permettant d'écrire des slides en Markdown et de les prévisualiser/export en temps réel. Différence vs RevealJS : syntaxe Markdown pure plutôt qu'HTML, export PDF/PPTX natif. Pour qui : développeurs et rédacte
