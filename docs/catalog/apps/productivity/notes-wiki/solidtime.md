# solidtime

> solidtime est une plateforme de suivi du temps pour équipes : projets, tâches, clients, membres, rôles, taux horaire, export, reporting, intégrations (via API). Stack Laravel + Vue 3, design moderne, mode "favoris" pour démarrer rapidement ses tâches récurrentes.
