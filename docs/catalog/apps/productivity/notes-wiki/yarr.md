# yarr

> yarr (« Yet Another RSS Reader ») est un lecteur de flux écrit par Nikita Kanaev en Go. Sa philosophie est radicale : pas d'authentification, pas de comptes, pas d'extensions, pas d'OAuth, pas de base de données externe.
