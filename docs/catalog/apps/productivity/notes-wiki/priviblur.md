# Priviblur

> Priviblur est un front-end alternatif pour Tumblr, développé dans la même philosophie que [[app-redlib]] pour Reddit ou [[app-invidious]] pour YouTube : proposer une interface légère qui interroge l'API publique de Tumblr en servant de proxy, tout en éliminant le tracking, les publicités et le JavaS
