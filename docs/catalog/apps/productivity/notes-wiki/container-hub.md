# Container Hub

> Container Hub est une interface web générique pour naviger dans un Docker Registry v2 / OCI registry. Elle liste repositories, tags, tailles, dates, et permet la suppression d'images. Différence vs Harbor UI: Container Hub est léger, mono-registre, sans scan ni RBAC — c'est un navigateur, pas une pl
