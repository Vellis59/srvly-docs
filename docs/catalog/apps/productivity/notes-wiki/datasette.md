# Datasette

> Datasette est un outil écrit par Simon Willison (célèbre blogueur et créateur de Django) qui transforme n'importe quel fichier SQLite en une API JSON explorable et une interface web de requêtage. Le principe : `datasette ma_base.db` et l'utilisateur dispose instantanément de filtres, tris, facettes,
