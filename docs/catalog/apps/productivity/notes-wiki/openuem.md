# OpenUEM

> OpenUEM est une plateforme de gestion unifiée des terminaux (UEM) : inventaire matériel/logiciel, déploiement d'apps, gestion de configuration, remote control, politiques de sécurité, chiffrement, supervision. Agents pour Windows, macOS, Linux. Console web centralisée.
