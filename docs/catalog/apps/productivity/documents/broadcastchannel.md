# BroadcastChannel

> BroadcastChannel est une application web de publication audio/podcast auto-hébergeable, conçue comme alternative minimaliste à Anchor, Substack, ou Transistor. L'idée : on upload un MP3, on ajoute une description, on génère automatiquement un flux RSS podcast (iTunes-compatible), et le tour est joué
