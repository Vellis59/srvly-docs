# Statamic

> Statamic est un CMS construit sur Laravel, qui mise sur le flat-file (contenu en fichiers YAML/Markdown versionnables dans Git) plutôt qu'une base de données. Il combine l'élégance éditorialiale de Ghost, la puissance de Laravel, et un Control Panel moderne (Vue.js) pour les auteurs. Différence vs W
