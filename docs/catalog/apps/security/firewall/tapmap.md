# TapMap

> TapMap est une application web qui permet de dessiner et maintenir une cartographie réseau précise, à la main ou semi-automatiquement. L'idée : remplacer les schémas Visio/Draw.io statiques par une carte interactive hébergée, qui peut afficher en temps réel l'état (up/down) de chaque équipement via
