# Meshping

> Meshping est un service de monitoring conçu spécifiquement pour les réseaux maillés (mesh networks) : il maintient une matrice N×N des latences ping entre tous les nœuds d'un cluster, et visualise l'évolution temporelle pour détecter les liens faibles, les routes qui se dégradent, ou les partitions
