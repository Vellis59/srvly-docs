# WGDashboard

> WGDashboard est une interface web self-hosted écrite en Python (Flask) qui permet de gérer plusieurs instances WireGuard simultanément depuis un point unique. Là où `wg` et `wg-quick` se pilotent en ligne de commande, WGDashboard apporte une UI claire, des graphes de trafic et des outils d'administr
