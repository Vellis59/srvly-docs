# Calorific

> Calorific est le pendant minimaliste de [[app-nutritrace|NutriTrace]] : un tracker qui ne suit que les calories, rien d'autre. Pas de macros, pas de micronutriments, pas de base alimentaire géante. Juste un compteur simple.
