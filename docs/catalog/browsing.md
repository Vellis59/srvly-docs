# Browsing the Catalog

The catalog page shows apps in a visual grid with:

- **App icon** and **name**
- **Short description**
- **Category pills** for filtering
- **Popular** and **New** sections

## Search

Use the search bar to find apps by name or description. The search is fast and indexes both title and description fields.

## Category Filter

Click on any category pill to filter apps. You can also browse dedicated category pages for a full view.
