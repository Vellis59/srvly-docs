# Browsing the Catalog

The catalog is organized as a **hierarchical tree** in the sidebar, making it easy to drill down from broad category to specific application.

## Sidebar Navigation

```
App Catalog
├── Overview
├── Browsing
├── Installing
├── Productivity
│   ├── Notes & Wiki (417 apps)
│   ├── Documents & Office (111 apps)
│   ├── Project Management (36 apps)
│   └── ...
├── Dev & Infrastructure
│   ├── Server Management (215 apps)
│   ├── Monitoring & Logging (98 apps)
│   └── ...
├── Communication
├── Media
├── Storage & Backup
├── Security
├── Automation & IoT
├── Gaming
├── AI & LLM
└── Finance
```

Click on any **category** (e.g., Productivity) to see its index page with all subcategories. Click on a **subcategory** (e.g., Notes & Wiki) to browse the full list of apps in that group.

## Category Index Pages

Each category has an automatically generated index page showing:
- A description of the category
- Links to all subcategories
- The number of apps in each subcategory

## App Detail Pages

Every app page provides:
- **Quick Info table** — category, subcategory, default HTTP port, Docker image, dependencies, supported OS
- **Links** to the official website, documentation, and source code
- **Tags** that link to related apps

## Search

Use the **search bar** at the top of the page to find apps by name, description, or tags. The search indexes all app pages and categories.

## Tags

Each app page includes tags (e.g., `productivity`, `notes-wiki`, `self-hosted`). Click on any tag to see all apps sharing that tag.

## Previous / Next Navigation

At the bottom of each app page, use the **Previous** and **Next** buttons to browse through apps in the same subcategory.
