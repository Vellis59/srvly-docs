# App Catalog

srvly provides access to **900+ open-source applications** ready for self-hosted deployment.

## Source

The catalog is sourced from the [vellis.cc](https://vellis.cc) directory — the largest French-language reference for open-source self-hosted software, with English descriptions for international users.

## Categories

Apps are organized into 10 parent categories with ~50 subcategories:

- **Infrastructure** — DNS, monitoring, logging, reverse proxy
- **Storage & Files** — cloud, sync, backup, NAS
- **Communication** — email, chat, VoIP, forums
- **Media** — streaming, books, podcasts, photos
- **Development** — CI/CD, git, containers, databases
- **Security** — VPN, auth, firewall, password managers
- **Personal** — bookmarks, notes, feeds, finance
- **Business** — CRM, ERP, analytics, project management
- **Gaming** — game servers, voice
- **Other** — home automation, IoT, misc
