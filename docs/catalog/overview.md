# App Catalog

srvly provides access to **1,668 open-source applications** ready for self-hosted deployment on your VPS.

## How to Navigate

Each category has its own index page listing all subcategories. Click into any subcategory to see the full list of available applications.

Every app page includes:
- **Quick Info** — category, subcategory, default port, Docker image, dependencies, OS support
- **Links** — official website, documentation, and GitHub repository
- **Tags** for cross-referencing

## Source

The catalog is curated from the open-source self-hosted software ecosystem and updated regularly. New applications are added as they become available.
