# Dashboard

The dashboard gives you a real-time overview of all your servers.

## Server Cards

Each server shows:

- **Status** — Online / Offline / Error with color indicator
- **System info** — OS, CPU cores, RAM (used/total), disk (used/total)
- **Uptime** — how long the server has been running
- **Health** — load average and network traffic
- **Quick actions** — logs, restart, stop/start, Docker cleanup

## Plan Banner

If you're on the Free plan, a banner shows your server usage (e.g., "1/1 servers used"). Self-hosted users see no limit.

## Navigation

- **Servers** — list and manage all your connected servers
- **Catalog** — browse 1,668+ open-source apps across 10 categories
- **Settings** — view your API token and server configuration
